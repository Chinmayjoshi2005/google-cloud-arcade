### Run the following Commands in CloudShell

### Fetch Project ID and Bucket Automatically
```bash
PROJECT_ID=$(gcloud config get-value project)
BUCKET=${PROJECT_ID}-bucket
```

### Update Content-Type for index.html (HTML Rendering)
```bash
gsutil setmeta -h "Content-Type:text/html" gs://${BUCKET}/index.html
```

### Update Content-Type for style.css (CSS Rendering)
```bash
gsutil setmeta -h "Content-Type:text/css" gs://${BUCKET}/style.css
```

### Update Content-Type for logo.jpg (Image Rendering)
```bash
gsutil setmeta -h "Content-Type:image/jpeg" gs://${BUCKET}/logo.jpg
```

### Enable Website Configuration on Bucket
```bash
gsutil web set -m index.html -e 404.html gs://${BUCKET}
```

### Make Bucket Public for Website Access
```bash
gsutil iam ch allUsers:objectViewer gs://${BUCKET}
```
